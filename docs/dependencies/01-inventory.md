## 5. CVE posture

This inventory is a **snapshot**. The live truth lives in
`/tmp/npm-audit.json` (or whatever `scripts/deps/audit.sh` writes to). Run:

```bash
./scripts/deps/audit.sh                  # markdown table, blocks CI on critical
./scripts/deps/audit.sh --json           # raw npm audit JSON
./scripts/deps/audit.sh --severity high  # high and above only
```

The audit script honours the `npm audit --production --json` contract
(see `npm` docs). For non-`npm` audit data (pnpm, yarn, OS package scanners),
pipe it through `scripts/deps/audit.sh --from-stdin`.
