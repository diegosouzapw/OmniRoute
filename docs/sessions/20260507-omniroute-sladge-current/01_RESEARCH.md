# Research

## Repository Fit

OmniRoute is a direct Sladge target. The README describes it as a free AI
gateway with smart routing to free and low-cost AI models, OpenAI-compatible
API behavior, MCP server support, A2A protocol support, and coding-agent
provider support.

## Existing Patterns

- The root README already starts with a badge block.
- Existing repo instructions require npm-based validation and Prettier for
  changed Markdown.
- Session docs live under `docs/sessions/<session-id>/`.

## Decision

Add the Sladge badge to the existing top-level badge block and keep the change
limited to README plus session governance docs.
