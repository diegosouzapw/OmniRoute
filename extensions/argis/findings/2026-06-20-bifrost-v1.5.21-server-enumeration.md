# Bifrost v1.5.21 SDK Break Enumeration

**Date:** 2026-06-20
**Purpose:** T1.2 — enumerate 9 Bifrost server plugins affected by v1.2.30 → v1.5.21 SDK bump.
**Refs:** ADR-050 §8 ACCEPTED (router Option B), v11 plan T1.1

## SDK break patterns (from prior T1.1 research)

Three known SDK breaks in Bifrost v1.5.21:

1. **MissingBatchPayloadKey** — `BatchPayload` struct now requires `RequestID` field (was optional in v1.2.30).
2. **stream-compat** — `StreamChunk` event signature changed from `(chunk []byte)` to `(chunk []byte, final bool)`.
3. **BifrostExecutor** — `Execute(ctx, req, opts...)` now returns `(*Response, error)` instead of `(*Response, metadata, error)`. Metadata moved to `Response.Meta`.
